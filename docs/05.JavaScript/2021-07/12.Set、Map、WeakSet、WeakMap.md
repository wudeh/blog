## Set
- Set对象可以存储任何类型的数据。 值是唯一的，没有重复的值。
- WeakSet 的成员只能是对象，而不能是其他类型的值；WeakSet 不可遍历；无重复值

## Map
- Map对象保存键值对，任意值都可以成为它的键或值。
- WeakMap 对象是一组键值对的集合，只接受对象作为键名（null除外）；键名所指向的对象为弱引用，不计入垃圾回收机制。
